HTTP Client
=======

&copy; 2018 SiLeader.

## Overview
Simple HTTP Client library

## Features
+ HTTP support
+ Only one header file
+ Support "Transfer-Encoding: chunked" and "Content-Length"
+ Support GET, HEAD, PUT, POST, DELETE method

## Dependencies
+ Boost C++ Libraries (Boost.Asio)
